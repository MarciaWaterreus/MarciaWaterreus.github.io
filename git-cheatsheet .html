Git and Powershell cheatsheet

A quick reference to all the basic commands for a Git VCS through Powershell.
Microsoft Powershell Docs

Basic Powershell commands

These are the basic cmdlets to work in Powershell.

CMDlet Name Description
E: drive Navigate to the correct drive
ls List List the directories for the current location
ls -la
ls -force
dir		 -la Does not work in Powershell! List the child items of the current directory: -la
-force displays hidden files as well as 'normal' files.
dir displays all child directories
cd Change directory Change to the specified directory
New-Item Create a new file

Git commands

CMDlet Name Description
git --version Git version Check the version of your Git.
git config --list Git configuration Check configuration settings for your Git installation.
git config --help
git help config		 Git action help Check the manual for the action you wish to know more about.
git clone https://... Clone the repository from the (Github) URL.
git remote add origin https://... Add a new empty repository as origin to push towards.
git remote -v Verifies the new remote URL.
git init Git initialize Create an empty Git repository in the specified location
rm -r -force .git
 Remove-Item -Recurse -Force Delete a (git) directory and it's files
git pull origin master Pull the master (and any changes that have been made since the last time) to your local repo.
git status Display the status for files to be tracked by Git
New-Item .gitignore Create a new (.gitignore) file
git add .gitignore
git add -A 		Add a specific file (or -A all the files) to the Staging area (changes to be committed)
git commit -m "Message here" Commit the changes to the repository, accompanied by a description
git reset .test.html
git reset		 Remove a file (or ALL files) from the Staging area (untracked status)
git log Display the unique SHA-hash for your Git commit, the user who committed it and timestamp.
git remote -v List information about the repository.
git branch -a List all branches in the repository.
git branch calc-divide
git branch branchname
git checkout -b branchname		 Create a new branch with a name of your choice (and start working from it immediately).
git branch Display the branch you are currently working on.
git diff Show changes that have been made to the code
git pull origin master Pull the changes that have been made since the last time to your local repo.
git push origin master
branchname		 Push the changes that have been made to your remote repo TO THE MASTER.
